# VALUEBOOKS LIFF買取モック

LINEで本棚整理 / LIFF買取の検討用モックです。

## GitHub Pagesで公開する手順

1. このフォルダ内の `index.html` を GitHub リポジトリのルートに置く
2. GitHubで `Settings` → `Pages`
3. `Build and deployment` を `Deploy from a branch` にする
4. Branchを `main` / `/root` にして保存
5. 数分後に以下のURLで公開されます

https://vbtaiki.github.io/liff/

## ローカル確認

`index.html` をブラウザで開くだけで確認できます。
